เหตุผลและแนวทาง

PrimePC เป็นระบบจัดการอุปกรณ์คอมพิวเตอร์ที่ออกแบบมาเพื่อช่วยให้สถาบันการศึกษาและทีม IT สามารถจัดสรร ติดตาม และดูแลอุปกรณ์ได้อย่างเป็นระบบ ปรับปรุงการใช้ทรัพยากรให้คุ้มค่ายิ่งขึ้น

วัตถุประสงค์ของโครงการ

พัฒนาแพลตฟอร์มจัดการอุปกรณ์พีซี ที่รองรับการจัดเก็บข้อมูล รายงานสถานะ และการติดตามการใช้งาน โดยสอดคล้องกับหลัก SDLC

ขอบเขตของระบบ

  • Admin, IT Department, ผู้บริหารระดับสูง และผู้ใช้ทั่วไป
  • จัดการอุปกรณ์ CPU, GPU, RAM และสถานะการใช้งาน
  • รองรับการค้นหา ดูข้อมูล และขอใช้งานอุปกรณ์

Analysis & Design

ระบบนี้ออกแบบให้มีใช้ฟีเจอร์หลักต่อไปนี้:

  • จัดเก็บข้อมูลอุปกรณ์และผู้ใช้งาน
  • โมดูลการจอง/จัดสรรอุปกรณ์
  • ระบบตรวจสอบสถานะอุปกรณ์และการซ่อมบำรุง
  • หน้าแสดงผลรายงานสรุปและแผนภูมิ

การออกแบบเน้นโครงสร้างที่ชัดเจน ระหว่างส่วนแสดงผล (Frontend) และส่วนจัดการข้อมูล (Backend)

System Architecture

ระบบแบ่งเป็น 3 ส่วนหลัก:

  1. Frontend: หน้าเว็บใช้งานของผู้ใช้
  2. Backend: บริการจัดการข้อมูลและ API
  3. Database: การเก็บข้อมูลอุปกรณ์และสถานะ

GitHub Pages

เอกสารนี้พร้อมให้เผยแพร่ผ่าน GitHub Pages โดยใช้โฟลเดอร์ docs/ บน branch main.

วิธีเปิดใช้งาน:

  1. ไปที่ GitHub repository > Settings > Pages
  2. เลือก Source เป็น main branch และ /docs folder
  3. กด Save เพื่อให้ระบบสร้างเว็บไซต์

URL ที่คาดว่าจะใช้งานได้:

https://gotbangsue.github.io/my-project-documentation/
